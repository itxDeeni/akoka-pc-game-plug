# Akoka PC Game Plug

A modern, full-stack digital storefront for game delivery.
- **Frontend:** Vue 3, Vite, Custom CSS (Dark Neon Gaming Theme)
- **Backend:** Node.js, Express, Axios, Cheerio (Dynamic Scraper)

## Features
- Dynamic size-based game pricing scraping
- WhatsApp-integrated cart delivery
- Fast, dark-mode responsive UI
